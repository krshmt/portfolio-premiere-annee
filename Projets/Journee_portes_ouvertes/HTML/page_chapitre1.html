  • LE HTML, C'EST QUOI ?

    Le langage HTML est un langage principalement utilisé pour la création de sites internet. Ce langage permet de structurer le contenu d'une page web. Grâce au HTML, nous pouvons créer des formulaires de saisie ou inclure du contenu visuel (vidéos, images, etc.).

    HTML est l'acronyme d'HyperText Markup Language. Il se traduit en « langage de balisage hypertexte ».

    L'HTML est souvent utilisé avec le langage de programmation JavaScript et/ou du CSS (que l'on verra plus tard dans ce MOOC)

  • QU'EST-CE QU'UN ÉLÉMENT HTML ?

    Un élément HTML est composé de plusieurs éléments :

    • Éléments HTML : Les éléments HTML sont utilisés pour structurer un document écrit en HTML.

    • Balise ouvrante/fermante : Une balise ouvrante/fermante permet de définir le début/la fin du contenu d’un élément HTML.

    • Attributs et valeurs : Certains éléments peuvent posséder des attributs, chacun associé à une valeur.

LES DIFFÉRENTES BALISES À SAVOIR :

BALISES DE TYPE BLOCK

Les balises de type block prennent toute la place disponible. Par exemple, un paragraphe constitué d'une seule ligne prendra toute la largeur de la page.

  • body représente le contenu principal du document HTML. Il ne peut y avoir qu’un élément body par document.

  • main représente le contenu ou la fonctionnalité principale du document.

  • header représente un groupe de contenu introductif ou de contenu aidant à la navigation. Il peut contenir des éléments de titre, mais aussi d’autres éléments tels qu’un logo, un formulaire de recherche, etc.

  • footer représente le pied de la section ou de la racine de sectionnement la plus proche. Un élément footer contient habituellement des informations sur l’autrice ou l’auteur de la section, les données relatives au droit d’auteur (copyright) ou les liens vers d’autres documents en relation.

  • aside représente une partie d’un document dont le contenu n’a qu’un rapport indirect avec le contenu principal du document. Les aside ("apartés" en français) sont fréquemment présents sous la forme d’encadrés ou de boîtes de légende.

  • h1-h6 représentent six niveaux de titres dans un document, h1 est le plus important et h6 est le moins important.

  • nav représente une section destinée à la navigation dans un document : des liens vers d’autres pages ou vers des fragments de cette page (menus, tables des matières, index, etc.).

  • section représente une section générique d’un document. Une section commence généralement avec un titre.

  • p représente un paragraphe de texte.

BALISES DE TYPE INLINE

Un élément de type inline ne prendra que la place qui lui est nécessaire pour s'afficher sur la page.

  • a (pour anchor en anglais), avec son attribut href, crée un lien hypertexte vers des pages web, des emplacements se trouvant dans la même page, etc.

  • code représente un court fragment de code machine.

  • img intègre une image dans le document.

BALISE DE TYPE LISTE

  • ul représente une liste non-ordonnée (puces, symboles).

  • ol représente une liste ordonnée (numérotée par défaut).

  • li est utilisé pour représenter un élément dans une liste. Il doit être contenu dans un élément parent : une liste ordonnée (ol), une liste non-ordonnée (ul ) ou un menu (menu).
