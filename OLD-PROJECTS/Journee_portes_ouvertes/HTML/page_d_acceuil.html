  • PRÉAMBULE DE NOTRE SITE INTERNET ET DE SES CRÉATEURS

    Nous sommes quatre étudiants de première année à l'IUT d'Orléans en BUT Informatique. Dans le cadre de notre formation, nous devons créer un cours en ligne (MOOC), portant sur le thème suivant : Comment faire créer un site web.

    Ce cours est composé de trois chapitres portant sur les bases fondamentales de l'HTML et le CSS et un quiz rapide pour voir ce que vous avez retenu. Ce cours est fait pour des personnes entièrement débutants dans le but de leur apprendre les bases. Il vise aussi à les rappeler aux personnes qui ont un niveau intermédiaire.

  • CRÉER VOTRE SITE WEB

    Vous souhaitez créer votre site web mais vous n’y connaissez rien ? Vous êtes au bon endroit !

    Dans ce cours, vous allez vous initier à la programmation avec des langages incontournables du Web : HTML5 et CSS3.

    C’est souvent par là qu’on commence quand on apprend à programmer, car ces deux langages sont à la base de tous les sites que vous pouvez voir lorsque vous naviguez sur internet !

    Si vous pensez ne pas avoir le niveau pour suivre ce cours, ne vous inquiétez pas : HTML et CSS sont des langages parfaits pour débuter. Nous allons les découvrir pas à pas : dans ce cours, on vous explique tout.

    Vous serez très vite capable d'ajouter du texte à votre site, de construire un menu de navigation, d'insérer des images... et bien plus encore !

    Alors, êtes-vous prêts à réaliser un site web de A à Z ? C’est parti !

  • LES PRÉREQUIS POUR COMMENCER VOTRE PREMIER SITE INTERNET

    Comme il est dit plus haut, vous n'avez pas besoin de certaines connaissances pour continuer ce cours. Les prérequis sont du côté de votre matériel. Pour commencer votre site WEB (chez vous par exemple), il vous faut

    • Un ordinateur (peu importe son système d'exploitation)
    • Un éditeur de code. Nous utilisons Visual Studio Code à l'IUT mais il y en a d'autres
    • Un navigateur internet pour pouvoir visualiser votre création (Chrome, Safari, Firefox ou encore Edge)

  • OBJECTIFS PÉDAGOGIQUES

    À la fin de ce cours, vous serez capable de :

    • Maîtriser les bases de HTML5.
    • Faire de la mise en forme avec CSS3.
    • Agencer le contenu des pages
